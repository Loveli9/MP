1.我想陪你看遍世间美景，然后告诉你，它们都不及你万分之一。早安！ 2.最好的爱情大概就是，我崇拜你像个英雄，你疼爱我像个孩子。早安！ 3.其实最好的日子，无非是你在闹，我在笑，如此温暖过一生。早安！ 4.今天将变成昨天，明天将变成从前，但我会一直陪在你身边，把现在变成永远。早安！ 5.你在我身边也好，在天边也罢，想到世界的角落有一个你，觉得整个世界也变得温柔安定了。早安！ 6.你是我毛茸茸的外套，冬日温暖的怀抱，大雪里的避风港，余生要牵手的意中人。早安！ 7.遇上你是今生的缘份，爱上你是我今生的幸福，守护你是我今生的选择，为你我今生无怨无悔，爱你一生不变！早安！ 8.我原是凡人一个，看山是山，看水是水，自从遇见了你，世间所有美好终成景象。早安！ 9.光阴荏苒，激情归于平淡，就让我陪你度过平凡。请你相信，只要我的心还在跳，它便是因你而跳。早安！ 10.我从未试过这样去对一个人好，就只是想纯粹的付出，甚至忘记了在乎一下回应。早安！ 11.不过一场相遇，也只是一不小心便记住了你，目光紧随你悲伤或欢喜都想告诉你。早安！ 12.其实全世界最幸福童话，不过是，与你一起度过柴米油盐岁月。早安！
